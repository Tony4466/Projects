Python Chess Engine

I developed a fully independent chess engine in Python, implementing custom board representation, move generation, search algorithms, and evaluation logic from the ground up. The engine supports human play, AI play, and performance analysis across multiple games.

The engine functions independently of external chess programs. The chess library and Stockfish are imported strictly for comparison and evaluation purposes (e.g., translating positions or benchmarking move quality), not for gameplay or decision-making.

Search & Optimization Techniques

  • Iterative deepening move search
  • Alpha–beta pruning
  • Aspiration (search depth) windows
  • Quiescent search for tactical stability
  • Move ordering heuristics
  • Killer move heuristic
  • Zobrist hashing
  • Transposition table
  • Opening database integration

Evaluation Function (Centipawns)

  • Material values (P=100, N=300, etc.)
  • Piece-square tables for positional value
  • Pawn structure analysis
    • Penalizes doubled and isolated pawns
    • Rewards passed pawns
  • Rewards rooks on open files
  • Rewards knights on strong outposts
  • Endgame heuristics (king activity, simplified positions)
